# Iconography

Source: https://learn.microsoft.com/zh-tw/windows/apps/design/signature-experiences/iconography

Source snapshot: Microsoft Learn zh-tw page reviewed on 2026-06-06.

Use icons to communicate concepts, actions, and states quickly.

## Guidelines

- Prefer Segoe Fluent Icons for system icons in WinUI command, navigation, and status surfaces.
- Keep icons minimal: include only the detail needed to communicate the concept.
- Keep icons harmonious: use simple geometric forms that fit Windows 11.
- Use modern, recognizable metaphors.
- Use 16 epx icons for common command and title bar contexts unless the surrounding control requires another standard size.
- When modifying a base icon, put modifier symbols in a lower quadrant and avoid obscuring the base metaphor.
- Use icon layering for states such as active or selected when appropriate.
- Validate cultural meaning; some symbols may need localization or replacement.
