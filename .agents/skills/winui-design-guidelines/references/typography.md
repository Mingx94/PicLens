# Typography

Source: https://learn.microsoft.com/zh-tw/windows/apps/design/signature-experiences/typography

Source snapshot: Microsoft Learn zh-tw page reviewed on 2026-06-06.

Typography communicates structure; style must not block comprehension.

## Guidelines

- Use Segoe UI Variable for Windows apps when available.
- Use one typeface family across the app unless a specific language or brand requirement forces otherwise.
- XAML scales type across devices; do not change font size based only on screen size or resolution.
- Design with effective pixels.
- Use hierarchy: titles summarize, body text explains.
- Use regular weight for most text and semibold for headings or emphasis.
- Align text left by default; center only in limited cases such as labels under icons.
- Minimums: 14 px semibold, 12 px regular. Smaller text can become unreadable in some languages.
- Use sentence-style capitalization for English UI. For Chinese/localized UI, follow locale conventions and keep wording natural.
- Use ellipsis for most truncation; clipping should be rare.

## Windows 11 Type Ramp

- Caption: 12/16 epx.
- Body: 14/20 epx.
- Body strong: 14/20 epx semibold.
- Body large: 18/24 epx.
- Body large strong: 18/24 epx semibold.
- Subtitle: 20/28 epx semibold.
- Title: 28/36 epx semibold.
- Title large: 40/52 epx semibold.
- Display: 68/92 epx semibold.
