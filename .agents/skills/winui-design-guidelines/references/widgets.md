# Widgets

Source: https://learn.microsoft.com/zh-tw/windows/apps/design/widgets/

Source snapshot: Microsoft Learn zh-tw page reviewed on 2026-06-06.

Windows widgets are small Adaptive Card surfaces associated with an installed app.

## Guidelines

- Widgets should be glanceable, reliable, useful, personal, focused, and fresh.
- A widget should not replace the app or website; it should surface the most important information or quick action.
- Avoid ads in widgets.
- Focus each widget on one primary task or scenario.
- Support small, medium, and large sizes when possible. The purpose should stay the same while larger sizes reveal more information.
- Use deep links so clicks take users directly to relevant app or web content.
- Refresh content based on context and time so the widget remains useful.
