# Color

Source: https://learn.microsoft.com/zh-tw/windows/apps/design/signature-experiences/color

Source snapshot: Microsoft Learn zh-tw page reviewed on 2026-06-06.

Use color to show hierarchy, structure, state, and emphasis without overwhelming the interface.

## Guidelines

- Respect light and dark themes; default to the user's Windows theme unless the app has a strong reason to override it.
- Use theme brushes/resources so background, text, icons, and controls adapt automatically.
- Use accent color sparingly for important elements and interaction states.
- Keep color meaningful. Choose one consistent color convention for interactive elements.
- Do not use color as the only distinction between states or categories.
- Check contrast in all themes and high contrast.
- Consider lighting conditions and color blindness. Avoid red/green as the only differentiator.
- Consider cultural interpretation when selecting semantic colors.
