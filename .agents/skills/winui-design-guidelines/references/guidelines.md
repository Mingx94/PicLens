# WinUI Design Guidelines Reference Map

Source snapshot: Microsoft Learn zh-tw pages reviewed on 2026-06-06. Refresh the linked pages when the user asks for latest guidance or when platform behavior may have changed.

Use this file as an index. Load only the topic file needed for the current task.

## Topic Files

- `overview.md` - overall principles and source snapshot context.
- `navigation.md` - app structure, `NavigationView`, `TabView`, breadcrumbs, list/details, and back behavior.
- `commanding.md` - buttons, command surfaces, direct manipulation, feedback, dialogs, confirmation, and undo.
- `layout.md` - effective pixels, breakpoints, responsive design, app silhouettes, and title bars.
- `color.md` - theme, accent color, contrast, color blindness, and cultural interpretation.
- `typography.md` - Segoe UI Variable, type ramp, sizing, alignment, and truncation.
- `geometry.md` - corner radii, connected edges, and global corner resources.
- `layering-and-elevation.md` - layers, elevation, shadows, strokes, and focus hierarchy.
- `materials.md` - Mica, Acrylic, and Smoke.
- `iconography.md` - Segoe Fluent Icons, modifiers, layering, and localization.
- `motion.md` - WinUI transitions, connected animations, timing, and easing.
- `usability.md` - accessibility, app settings, localization, and help.
- `widgets.md` - Windows widget principles and Adaptive Card surfaces.
- `writing-style.md` - UI copy, errors, dialogs, and button labels.
- `review-checklist.md` - final WinUI design review checklist.
