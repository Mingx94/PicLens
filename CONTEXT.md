# PicLens

PicLens is a desktop image organizer and viewer for local folders. This context names the product concepts used when describing browsing, selection, file organization, and preview behavior.

## Language

**Library**:
The currently browsed collection of folder and image entries under the user's chosen folder.
_Avoid_: Gallery, catalog

**Folder Tree**:
The navigable folder hierarchy rooted at the user's chosen folder.
_Avoid_: Directory tree, sidebar tree

**Library Item**:
A folder or image entry shown in the library.
_Avoid_: Tile, grid item

**Selected Image**:
An image in the library that is marked for an action such as rename, trash, conversion, or drag rename.
_Avoid_: Picked image, active image

**Safe File Action**:
A user-requested file change that avoids overwriting and destructive deletion, and reports success, skipped, and failed results.
_Avoid_: File operation, batch job

**Image Viewer**:
The in-window view for inspecting one image from a fixed image sequence.
_Avoid_: Preview window, lightbox
